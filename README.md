# ImóvelPrime — Landing Page de Imobiliária Premium

Landing page profissional para consultoria imobiliária, desenvolvida por **Vinicius Felipe** como projeto de portfólio. Demonstra habilidade em criar sites institucionais complexos com múltiplas interações JavaScript.

## 🔗 Demo
> Após o deploy: `https://seu-usuario.github.io/imovel-prime`

## Sobre o Projeto
Landing page completa para uma imobiliária de alto padrão, com busca de imóveis, grid filtrável de propriedades, contadores animados de estatísticas e formulário de contato.

## Tecnologias
- HTML5 Semântico
- CSS3 (Custom Properties, Grid, Flexbox, Transitions)
- JavaScript Vanilla (ES6+)
- SVG inline para ícones e logo

## Funcionalidades
- Navbar com logo SVG, número de telefone e menu mobile (hambúrguer → X)
- Hero com formulário de busca de imóveis (tipo, bairro, quartos, preço)
- Abas no formulário: Comprar / Alugar / Lançamentos
- **Barra de estatísticas** com contadores animados (Intersection Observer)
- **Grid de imóveis** com filtro por tipo (Apartamento / Cobertura / Casa)
- Botão de favoritar em cada card (toggle ♡ → ♥)
- Tags de status por imóvel (À Venda / Exclusivo / Aluguel)
- Seção de 4 serviços (compra, venda, locação, consultoria)
- Seção "Sobre" com floater de credenciais
- Depoimentos de clientes
- Seção de contato com canais (telefone, WhatsApp, e-mail)
- Formulário completo com select de interesse e textarea
- Botão flutuante do WhatsApp
- SEO completo: meta description, Open Graph, Twitter Card, canonical URL
- Máscara de telefone em JavaScript

## Deploy no GitHub Pages

### Passo a passo:
1. Crie um repositório chamado `imovel-prime`
2. Na pasta do projeto:
   ```bash
   git init
   git add .
   git commit -m "feat: landing page ImóvelPrime"
   git branch -M main
   git remote add origin https://github.com/SEU-USUARIO/imovel-prime.git
   git push -u origin main
   ```
3. No GitHub: **Settings → Pages → Source: main / root → Save**
